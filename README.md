#### Game 'Sudoku'
Entry point ```main.py``` <br>
**Usage** <br>
```e``` - exit <br>
```n``` - new game (fill randomly some cells) <br>
```xy v``` - fill cell <br>
**where** <br>
```x``` - x-axis coordinate <br>
```y``` - y-axis coordinate <br>
```v``` - value (1-9)
